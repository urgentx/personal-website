This website

This website

A forray into web-design has brought to life this very website. I discovered the powerful layout pairing of HTML/CSS, while experimenting with dynamic content using Javascript/JQuery. Implemented libraries and plug-ins including Lightbox, SlidesJS, and Material Design Lite. The website adapts to various devices' screen size, and makes use of the Web Storage protocol to handle persistence. Performance is improved using browser caching and asynchronous Javascript loading.

Black Metal

Black Metal

An Android app with appeal to cultural niche. Takes a picture and applies image processing and text-overlay based on settings chosen. The app implements up-to-date layout elements and custom Views, giving it a professional, clean look. Divided simply into two Activities, it works quickly and is intuitive, with a Tab selector enabling Fragments to display more information on the one page. Image processing algorithms are drawn from my Graphics course at university and adapted to suit this style of art, and the user is given the option of going social, with a share button for Facebook, as well as the option to save to the device's Media Gallery.

Neuron

Neuron

Hi-fidelity website prototype for a home automation system, done for a university assignment. Uses Material Design Lite for components and is built upon a Lo-Fidelity prototype study. Includes supporting design decision documentation (colour scheme, users and task description, etc.) as well as wire-frame models made using Pencil. The emphasis here was on a simple and minimalistic approach, a warm, strong colour scheme and purely front-end model with login/logout functions ommitted.

Demon Rush

Demon Rush

Simple 2D platformer written in Java with multiple levels and enemies. Features a screen shake, soundtrack, and wondrous programmer art. I learned to keep projectile speeds relative to only screen co-ordinates, and how to efficiently check for collisions. It's also ported to Android, although while it runs well on the PC, many older mobiles experience performance issues thanks to the way assets are loaded in Android. A remake using either a custom asset loader or a library such as LibGDX could be a possibility.
